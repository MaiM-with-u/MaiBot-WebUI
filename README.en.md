### Getting Started

###### Prerequisites

1. Nodejs 20.0+
2. Python 3.12+

###### **Installation**

1. Clone the repo

```sh
git clone https://github.com/MaiM-with-u/MaiBot-WebUI.git
```

2. 
```sh
# Install pnpm
npm install pnpm

# Install dependencies
pnpm i

# Run development server
pnpm run dev

# Build for production
pnpm run build
```
