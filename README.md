### 上手指南

###### 开发前的配置要求

1. Nodejs 20.0+
2. Python 3.12+

###### **安装步骤**

1. Clone the repo

```sh
git clone https://github.com/MaiM-with-u/MaiBot-WebUI.git
```

2. 
```sh
# 安装pnpm
npm install pnpm

# 安装依赖
pnpm i

# 运行开发服务器
pnpm run dev

# 运行构建
pnpm run build
```
